CASE STUDY // PC PLATFORMER PROTOTYPE

Puss in Boots: The Last Wish (2022)

SITUATION & TASK

Developed as a final semester project for an introductory Unity course. While the baseline prompt challenged students to create a standard physics-based Monkey Ball clone, I pivoted the scope to design a custom third-person precision platformer based on a cinematic sequence from Puss in Boots: The Last Wish.

ROLE & SYSTEM

Sole Developer. Responsible for the fundamental gameplay mechanics, input programming, level layout architecture, and aesthetic lookdev. Scripted the hazard detection for the lava field, designed the character interaction triggers, and integrated the victory state system upon reaching the ending landmark.

TECH STACK

Unity, C# Scripting, Character Controller Physics, Animator Controllers, 3D Platforming Logic, Level Design Layout, UI HUD Setup.

PROJECT INFO

Duration: End of Semester Project.

Context: Unity 101 Course.

Humanoid Movement & Animator Architecture

Pivoted from the standard physics-based ball mechanics required by the course assignment to implement a full Humanoid Character Controller. The core technical challenge was mapping out the custom movement logic for a bipedal character, alongside structuring a clean Animator Controller. I focused on managing smooth blend trees and state transitions to correctly synchronize inputs—implementing standard isGrounded check logic to strictly prevent accidental double-jumps—and ensuring all states flash fluidly with the character's active 3D animations.

Narrative Mechanics & Single-Life Constraint

Integrated thematic components directly into the gameplay loop by enforcing a strict single-life constraint, mirroring the film's narrative of Puss having only 1 life left. I established custom trigger-volumes spanning across the active lava hazard field. Dropping below the stone pillars immediately trips a defeat condition and resets the scene, forcing the player to master the basic platforming layout to successfully reach the companion NPC at the level's exit.
