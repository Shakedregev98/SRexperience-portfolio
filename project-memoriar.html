CASE STUDY // MOBILE AR

MemoriAR (2023)

SITUATION & TASK

Following October 7th, 2023, there was a profound need for public commemoration when physical monuments couldn't be built. Developed for Google's Immersive Geospatial Challenge, the task was to create a rapid mobile AR prototype that allows anchoring digital monuments in meaningful locations.

ROLE & SYSTEM

Co-Developer & 3D Tech Artist (Team of 3). Responsible for the AR integration pipeline, implementing geospatial positioning, designing spatial gameplay logic, creating the 3D assets, and configuring mobile-optimized visual effects.

TECH STACK

Unity, Google Geospatial API, Cesium for Unity, VFX Graph, Custom Shading (Edge/Hologram Shader), Mobile AR (Android/iOS), 3D Modeling (Autodesk Maya).

PROJECT INFO

Duration: 1 Month.

Context: Google's Immersive Geospatial Challenge .

Geospatial AR Alignment

Integrated Google's Immersive Geospatial API with the Cesium plugin inside Unity. Developed the logic to map real-world GPS coordinates into runtime Unity world space, ensuring precise anchoring of digital monuments in public spaces.

VFX & Custom Shading

Developed a custom holographic edge shader to give the main digital monument its solemn, ethereal look. Combined the shader with a stylized particle network inside Unity's VFX Graph to simulate dynamic, flowing lines of light that form the monument's structure in real time.

Mobile AR Prototyping & UI

Designed a clean mobile interface prioritizing real-world user interaction. Engineered the system logic to handle real-time inputs—allowing users to seamlessly select and cast interaction triggers (like placing a candle or writing messages) within a controlled spatial radius around the monument.
